Populate Data

Un servicio para acceder a cientos de datasets públicos de forma unificada

Consume datos de cientos de fuentes públicas de forma unificada

Accede en tiempo real via API, o prepara y descarga en CSV

Datos siempre actualizados

Populate Data es un agregador de datos públicos y abiertos que ofrece un interfaz único de acceso a datos de fuentes diversas, unificando la forma en la que se recuperan y simplificando su integración en sistemas propios.

Con Populate Data no tendrás que malgastar tiempo descargando datos de fuentes y formatos diversos (integrarte con una API, descargar un CSV, parsear un PDF…): Populate Data se encarga de estas tareas y te ofrece un acceso único.

Casos de uso

Comparador de presupuestos municipales

Herramienta de comparación y análisis de los presupuestos de los +8.000 municipios españoles. Se puede visualizar el presupuesto de un municipio; configurar rankings por partida, filtrando por distintos criterios; así como crear comparaciones de partidas entre distintos municipios. Todos los datos se extraen en tiempo real de la API de Populate Data.

Visitar.

España en Cifras

Especial para las elecciones municipales y autonómicas de 2016, en el que se compara la situación de un municipio, provincia y autonomía a través de sus principales indicadores económicos, demográficos y de empleo. Los datos se preproceseraon en un índice de Elastic Search desde la API de Populate Data.

Visitar.

Observatorio Municipal

Módulo de Gobierto (plataforma de transparencia para administraciones públicas) que permite explorar indicadores estadísticos sobre un municipio.

Visitar.
